# 🌍 Air Quality Prediction

## 📌 Project Overview

This project focuses on predicting air quality levels using machine learning techniques.  
The goal is to analyze environmental data and build predictive models that estimate air pollution levels based on different environmental factors.

Air quality prediction is important for monitoring environmental conditions and helping authorities make informed decisions regarding pollution control and public health.

---

## 🎯 Project Objectives

- Analyze air quality data to understand pollution patterns
- Clean and preprocess the dataset
- Perform exploratory data analysis (EDA)
- Visualize environmental trends
- Build machine learning models to predict air quality
- Evaluate model performance and select the best model

---

## 📊 Dataset

The dataset contains environmental variables that affect air quality such as:

- Temperature
- Humidity
- Wind Speed
- Gas Concentrations
- Other Environmental Indicators

These features are used to predict the **Air Quality Index (AQI)**.

## ⚙️ Project Workflow

### 1️⃣ Data Cleaning

- Formatting data

### 2️⃣ Exploratory Data Analysis (EDA)

### 3️⃣ Data Visualization

Charts used in the project include:

### 4️⃣ Feature Engineering

Selecting the most important features that influence air quality.

### 5️⃣ Model Training

Several machine learning models were tested:

- Linear Regression
- Decision Tree
- Random Forest

### 6️⃣ Model Evaluation

Models were evaluated using:

- MAE
- MSE
- R² Score
